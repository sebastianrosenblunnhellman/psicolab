# Estadística con R para Psicología

Este curso te guiará paso a paso en el uso de R y RStudio para el análisis de datos en psicología. Desde la instalación hasta la realización de pruebas estadísticas complejas y la visualización de resultados.

Aprenderás a:
* Instalar y configurar el entorno de trabajo.
* Manipular y limpiar bases de datos.
* Realizar análisis descriptivos e inferenciales.
* Crear gráficos de calidad de publicación.
